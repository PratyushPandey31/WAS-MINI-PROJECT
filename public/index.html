Bcrypt Password Shield

OWASP A02: Cryptographic Failures

Passwords undergo irreversible salt-hashing using 12 cryptographic rounds. Eliminates plaintext database exposure and renders offline dictionary attacks mathematically infeasible.

Stateless Session Tokens

OWASP A01: Access Control Audit

Secures web sessions via signed JSON Web Tokens (JWT) using HMAC-SHA256. Restricts token reading through HttpOnly guidelines to prevent cross-site script (XSS) extraction.

TOTP Multi-Factor Gate

OWASP A07: Authentication Security

Secondary validation layer requiring rolling 30-second time-based OTP codes (RFC 6238). Google Authenticator integrations block unauthorized logins even during password theft.

Containerized Isolation

Virtualization Sandbox

Fully containerized service using minimal Alpine Linux layers. Configured with a restricted, non-root user account inside the container to prevent privilege breakouts.

SIEM Audit Log Streams

Compliance Auditing

Generates structured JSON records tracking authorization attempts, rate limits, and enrollment spikes, ready for streaming into cloud auditors like CloudWatch or Splunk.

DDoS Attack Rate Limiter

Threat Vector Defense

Mitigates brute-force scripts at the network perimeter. Rapid authentication requests automatically trigger temporary IP bans via status code 429 (Too Many Requests).
